# Tic Tac Toe (React)

A simple **Tic Tac Toe game built with React**

## How to Play

1. Start the game.
2. Player **X** goes first.
3. Players take turns selecting an empty cell.
4. The game ends when:
   - A player gets 3 in a row, or
   - All cells are filled and the game is a draw.

## Install dependencies

```bash
npm install
```

## Start Game

```bash
npm run dev
```
